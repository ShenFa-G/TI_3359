* To make a bootable sdcard (Windows host):

  1.Use the "HP USB Disk Storage Format Tool" to format your uSD card to FAT32

  2.Copy all the file in folder "BIN" to your uSD card.
    Then the system can boot up.
    The foder CN means Chinese version NK file,EN means English version NK file
    You can choose any one NK file to boot up.

  3.The fisrt time system boot up,it need to format the NAND FLASH,
    This process will need about 3 minutes.
    You can monitor this process from the Serial Output.
